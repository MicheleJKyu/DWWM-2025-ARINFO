Tableau 3 : Balises de structuration du contenu d’une page web.
Nom Description Type de la balise Balise auto fermante Attributs principaux obligatoires
<blockquote> Structure une section de contenu qui est citée d'un autre document ou d'une autre source. Block Non cite, p, q
<abbr> Représente une abréviation ou un acronyme. Inline Non -
<strong> Indique que le texte a une importance particulière ou un certain sérieux voire un caractère urgent. Inline Non -
<small> Permet de représenter des commentaires ou des textes à écrire en petits caractères. Inline Non -
<h1,h2,h3,h4,h5,h6> Balises titres qui permettent de hiérarchiser la structure d'un texte sur le web. Inline Non -
<figure> Représente un contenu autonome, éventuellement accompagné d'une légende facultative, qui est spécifiée à l'aide de l'élément "figcaption". Souvent lié à une image ou un extrait de code. Block Non figcaption, blockquote, cite, pre, p,
<figcaption> Représente une légende ou une légende décrivant le reste du contenu de son élément parent "figure", fournissant à "figure" une description accessible. Inline Non -
<img> Sert à intégrer des images dans une page Web. Inline Oui
<svg> Un fichier SVG est un type de format utilisé pour définir des graphiques bidimensionnels pour le web et basé sur XLM. Inline Oui -
<audio> Permet d'intégrer du contenu sonore dans un document, comme de la musique ou d'autres flux audio. Inline Non
<video> La balise <video> permet d'intégrer du contenu vidéo dans un document, comme un extrait vidéo ou d'autres flux vidéo. Inline Non -
<a> <a> (pour ancre ou anchor en anglais), avec son attribut href , crée un lien hypertexte vers des pages web, des fichiers, des adresses e-mail, des emplacements se trouvant dans la même page, ou tout ce qu'une URL peut adresser. Inline Oui -
<p> Permettent de définir un paragraphe. Block Non -
<br> Permet un retour à la ligne (jusqu'à la marge gauche) sans création de nouveau paragraphe. Inline Oui -
<hr> Cette balise permet de tracer un trait de séparation dans le texte. Inline Oui -
<del> Représente une portion de texte ayant été supprimée d'un document. Inline Non -
<progress> Indique l'état de complétion d'une tâche. Block Non id, max, value
<iframe> Élément HTML qui permet d'intégrer un contenu externe dans une page web. Block Non id, title, width, height, scr
<ul> Représente une liste d'éléments sans ordre particulier. Block Non -
<li> Permet de passer à l'élément suivant dans une liste. Block Non -
<ol> Représente une liste ordonnée. Block Non -
<dl> Fournit la description, la définition ou la valeur du terme précédent "dt" dans une liste de description. Block Non dt, dd
<dt> Identifie un terme dans une liste de définitions ou de descriptions. Inline Non -
<dd> Fournit la description, la définition ou la valeur du terme précédent "dt" dans une liste de description "dl". Inline Non -

Tableau 6 : Balises de construction d’un formulaire.
Balise Description Type de la balise Balise auto fermante Attributs principaux obligatoire
<form> Représente un formulaire, consistant en des contrôles interactifs permettant à l'utilisateur d'envoyer des données à un serveur web. Block Non action, method
<input> Permet à l'utilisateur de saisir des données. Inline Oui type, name
<textarea> Permet à l'utilisateur de saisir des données multi-lignes. Block Non Name
<button> Permet à l'utilisateur d'activer un contrôle ou d'envoyer un formulaire. Inline Non -
<select> Permet à l'utilisateur de sélectionner une ou plusieurs options. Inline Non -
<option> Représente une option dans un élément "select". Inline Non -
<label> Définit une étiquette pour un élément "input". Inline Non for
<fieldset> Permet de regrouper des éléments d'un formulaire avec une légende. Block Non -
